# French Accelerated

A simple web application to help learn French pronunciation of letters and numbers.

## Features

- **Two Tabs**: 
  - Letters: A-Z alphabet
  - Numbers: 0-20 in French
  
- **Audio Pronunciation**: Tap/click any letter or number to hear its French pronunciation using the browser's built-in text-to-speech (Web Speech API)

- **Visual Feedback**: Buttons animate when speaking to provide clear feedback

- **Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Use

1. Open `index.html` in a modern web browser (Chrome, Firefox, Safari, or Edge)
2. Click on the "Letters" or "Numbers" tab to switch between them
3. Tap or click any letter or number to hear its French pronunciation

## Technology

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (ES6+)
- Web Speech API for text-to-speech functionality

## Browser Compatibility

This application requires a browser that supports the Web Speech API:
- Chrome 33+
- Firefox 49+
- Safari 14.1+
- Edge 14+
